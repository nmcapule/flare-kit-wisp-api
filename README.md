A batteries included template for kick starting a TypeScript Cloudflare worker project.

## 🔋 Getting Started

### Running a devserver

```
$ npm run dev
```

### Deploying to Cloudflare Workers

```
$ npm run deploy -- --env <env>
```

### Uploading secrets from .secrets.env

```
$ make upload_secrets
```
